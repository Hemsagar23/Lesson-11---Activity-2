What is the Box Model?

The CSS box model is a layout principle that wraps every HTML element with content, padding, border, and margin.

This image sits inside a styled box with border and padding applied.

Example CSS Code:

.box {
  width: 300px;
  padding: 20px;
  border: 5px solid #333;
  margin: 50px auto;
  background-color: #4caf50;
  color: white;
}
